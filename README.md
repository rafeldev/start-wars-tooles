# start-wars-tooles - Prueba tecnica Frontend

## Built with 🛠️

- React
- Chakra UI
- Vite

## Installation 🔨

1. **Clone Project**: 
```
git clone https://github.com/rafeldev/start-wars-tooles.git
```

2. **Install dependencies**: 
```
npm install
```

3. **execute program in development mode**: 
```
npm run dev
```
